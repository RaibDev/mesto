# Проект: Место
#### Функциональность проекта позволяет использовать попап: вносить в нем изменения записи пользователя, лайкать понравившиеся карточки мест, добавлять новые карточки.
##### Технологии проекта:
* HTML5
* CSS3
* JS
* Figma
* Git
* BEM Nested
