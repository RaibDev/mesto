# Проект: Место
## Функциональность проекта позволяет использовать попап, вносить в нем изменения записи пользователя. Далее будет реализована возможность лайкать понравившиеся карточки мест, добавлять свои, менять аватар
### Технологии проекта:
* HTML5
* CSS3
* JS
* Figma
* Git
* BEM Nested
